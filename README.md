# Trendies.fyi

Compare stocks to things like Google Trends (which is currently the only thing supported), more to come soon!

## Features

- Ability to view Stocks and Google Trends
- 2 different views, combined and split view
- Sharable links (state is stored in URL props!)
- Ability to set colors on a per line basis
- OG images! These should include a screenshot of the graph (but this is still buggy and delayed)
- The ability to bring a smile to your face (results vary on this one!)

## Architecture

- Next.js (framework)
- Vercel (for hosting)
- Vercel KV (for rate limiting)
- [AlphaAdvantage](https://www.alphavantage.co/) for Stocks
